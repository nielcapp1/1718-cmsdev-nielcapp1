Promo Website

Niet-functionele specificaties

De website bevat meerdere pagina’s in het CMS. De inhoud van deze pagina’s bestaat uit informatie en promotie voor de Mobile Web App.

Voor de backoffice doen we beroep op een standaard Drupal 8 Admin Theme. De frontoffice bestaat uit een zelfgemaakt Custom Theme.

Aan de hand van een RESTfull API kan de Mobile Web App beroep doen op informatie van het Content Management System.

Functionele specificaties

Op de website worden de algemene voorwaarden geplaatst.

Op de FAQ pagina kan je de volgende vragen raadplegen:

  • Hoe schade doorgeven?
  • Hoe terugbrengen?
  • Waar zijn er laadpunten te vinden?

De website bevat naast tekst ook foto’s.
